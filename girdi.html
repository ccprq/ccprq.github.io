Kullanıcıdan Girdi Alma

  • C# da kullanıcıdan girdi almak için : Console.ReadLine(); kullanılır.

  • Kullanım şekli aşağıda verilmiştir. 
    
class Program
{
    static void Main(string[] args)
    {
        string isimsorgu = "Lütfen isim giriniz : ";
        
        Console.WriteLine(isimsorgu);

        string cevap = Console.ReadLine();

        Console.WriteLine(cevap);

        // WriteLine kullandığımız için girdiyi sorunun yanına değilde altına yazdırdı, Write methodu kullanırsak yanına yazdırabiliriz.
        Console.Write(isimsorgu);

        string cevap2 = Console.ReadLine();

        Console.WriteLine(cevap2);

        Console.ReadKey();
    }
}
