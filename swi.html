Switch Case

  • Switch Case yapısı, sabit değerlerle çalışır. Yani if, else, else if de olduğu gibi karşılaştırma yapılamaz. switch(){}, case:, break;, defualt: kullanılır.

    • switch(){} : Parantezin arasına kontrol edilecek değer girilir. Süslü parantezin içine ise diğer işlemler yazılır.

    • case: : switch in parantezine girilen değeri case kontrol eder. Eğer true döndürse yani case e girilen değer elde edilirse altındaki kodlar çalışır.

    • break; : case: in altına yazılan komutlar çalıştıktan sonra break; komutu ile durdurulur. break; döngüleri ve bunun gibi olayları sonlandırmak için kullanılır.

    • defualt: : Belirtilen değerlerden hiç biri eşleşmezse çalışacak kodlar altına yazılır.

  • Kullanım şekilleri aşağıda verilmiştir. 
    
class Program
{
    static void Main(string[] args)
    {
        Console.Write("Lütfen bir gün numarası girin (1-7 arası):");
        int gunNo = Convert.ToInt32(Console.ReadLine());

        string gunAdi;
        switch (gunNo)
        {
            case 1:
                gunAdi = "Pazartesi";
                break;
            case 2:
                gunAdi = "Salı";
                break;
            case 3:
                gunAdi = "Çarşamba";
                break;
            case 4:
                gunAdi = "Perşembe";
                break;
            case 5:
                gunAdi = "Cuma";
                break;
            case 6:
                gunAdi = "Cumartesi";
                break;
            case 7:
                gunAdi = "Pazar";
                break;
            default:
                gunAdi = "Geçersiz gün numarası!";
                break;
        }

        Console.WriteLine("Girilen gün numarasının karşılığı: " + gunAdi);
        Console.ReadKey();
    }
}
