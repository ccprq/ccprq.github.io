QuickFix Programming Language Comprehensive Guide

QuickFix is a versatile programming language designed for quick scripting and automation tasks. This guide covers all the commands and features of QuickFix with examples and explanations.

1. Variables and Data Types

QuickFix supports three main data types: strings, integers, and booleans.

*myString "Hello, World!"
*myNumber 42
*myBoolean true
Variables in QuickFix are prefixed with an asterisk (*).

2. Output Commands

2.1 print

print -l -s "Hello, World!"
print -d -i *myNumber
print -l -b *myBoolean
-l: print with a new line, -d: print without a new line
-s: string, -i: integer, -b: boolean

2.2 beep

beep[1000;500]
Produces a beep sound. Parameters: frequency (Hz) and duration (ms).

3. Input Commands

3.1 read

read *userInput
Reads user input and stores it in the specified variable.

4. Control Flow

4.1 if, else if, else

if[*age > 18]
    print -l -s "Adult"
else if[*age == 18]
    print -l -s "Just turned 18"
else
    print -l -s "Minor"
fi

4.2 for

for[*i;0 < 5;++]
    print -l -i *i
rof

4.3 loop (while)

*counter 0
loop
    print -l -i *counter
    add 1 to *counter
    if[*counter == 5]
        break
    fi
end

5. File and Directory Operations

5.1 mkfil (Create File)

mkfil "example.txt"

5.2 mkdir (Create Directory)

mkdir "new_folder"

5.3 rfil (Read File)

rfil "example.txt" to []*fileContent

5.4 del (Delete File or Directory)

del "example.txt"

5.5 mfil (Move File)

mfil "source.txt" to "destination.txt"

5.6 mdir (Move Directory)

mdir "source_folder" to "destination_folder"

6. Array Operations

6.1 add (Add to Array)

[]*numbers
add 42 to *numbers

6.2 Array Access

print -l -i *numbers{0}

7. String Operations

7.1 String Concatenation

*fullName *firstName + " " + *lastName

7.2 len (String Length)

*strLength *myString.len

8. System Commands

8.1 cmd (Execute Command)

cmd["dir"] *result

8.2 beroot (Run as Administrator)

beroot

9. Console Manipulation

9.1 clear (Clear Console)

clear

9.2 title (Set Console Title)

title "My QuickFix Program"

9.3 icon (Set Console Icon)

icon "path/to/icon.ico"

9.4 cursor (Set Cursor Position)

cursor[10;5]

9.5 windowsize (Set Window Size)

windowsize[80;24]

10. Miscellaneous Commands

10.1 import (Import Script)

import "other_script.qf"

10.2 delay (Pause Execution)

delay 1000

10.3 convert (Type Conversion)

convert *myVar to -i

10.4 thread (Create Thread)

thread myThread
    print -l -s "This runs in a separate thread"
endt

t.myThread.start

Conclusion

As you become more familiar with the language, you can combine these commands to create powerful scripts and applications.
