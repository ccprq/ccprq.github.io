if, else, else if Karar Yapıları

  • C# da karar vermek veya karşılaştırma yapmak için : if, else, else if kullanılır.

    • if : Eğer anlamına gelir, içine yazıldığı koşul gerçekleşirse çalışır.

    • else : Başka analıma gelir, if koşulu gerçekleşmezse çalışır.

    • else if : if koşulu gerçekleşmezse çalışır. else den farkı ise if gibi parantezi vardır ve koşul gerçekleşmezse kendi koşulunu uygular.

  • Kullanım şekilleri aşağıda verilmiştir. 
    
class Program
{
    static void Main(string[] args)
    {
        // Kullanıcıdan 3 tane sayı alalım ve 'if' kullanarak büyükten küçüğe sıralayalım
        //Convert.ToInt32() methodu kullanıcının girdisini sayı türüne çevirir

        Console.Write("Sayı 1 : ");
        int sayi1 = Convert.ToInt32(Console.ReadLine());
        Console.Write("Sayı 2 : ");
        int sayi2 = Convert.ToInt32(Console.ReadLine());
        Console.Write("Sayı 3 : ");
        int sayi3 = Convert.ToInt32(Console.ReadLine());

        if (sayi1 > sayi2 && sayi1 > sayi3 && sayi2 > sayi3)
        {
            Console.WriteLine(sayi1.ToString() + ">" + sayi2.ToString() + ">" + sayi3.ToString());
        }
        else if (sayi1 > sayi2 && sayi1 > sayi3 && sayi2 < sayi3)
        {
            Console.WriteLine(sayi1.ToString() + ">" + sayi3.ToString() + ">" + sayi2.ToString());
        }
        else if (sayi1 < sayi2 && sayi2 > sayi3 && sayi1 > sayi3)
        {
            Console.WriteLine(sayi2.ToString() + ">" + sayi1.ToString() + ">" + sayi3.ToString());
        }
        else if (sayi1 < sayi2 && sayi2 > sayi3 && sayi1 < sayi3)
        {
            Console.WriteLine(sayi2.ToString() + ">" + sayi3.ToString() + ">" + sayi1.ToString());
        }
        else if (sayi1 < sayi3 && sayi2 < sayi3 && sayi1 < sayi2)
        {
            Console.WriteLine(sayi3.ToString() + ">" + sayi2.ToString() + ">" + sayi1.ToString());
        }
        else
        {
            Console.WriteLine(sayi3.ToString() + ">" + sayi1.ToString() + ">" + sayi2.ToString());
        }
        Console.ReadKey();
    }
}
