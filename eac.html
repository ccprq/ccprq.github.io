Foreach

  • foreach döngüsü, bir koleksiyonu veya bir diziyi sırayla gezinmek için kullanılır. Bu döngü, her bir elemanı belirtilen koleksiyondan alır ve döngü gövdesinde belirtilen işlemleri gerçekleştirir.

  • Kullanım şekilleri aşağıda verilmiştir. 
    
    class Program
    {
        static void Main(string[] args)
        {
            int[] dizi = { 1, 2, 3, 4, 5 };

            foreach (int eleman in dizi)
            {
                Console.WriteLine(eleman);
            }
            Console.ReadLine();
        }
    }
