Ekrana Yazdırma

  • C# da console ekranına yazı yazdırmak için 2 method kullanılır. Bunlar : Console.Write();, Console.WriteLine();

    • Console.Write(); : Alt alta yazıldığında çıktı yan yana olur. Bu durumu bitirmek için yan yana yazım işlemi bittiğinde sonuncu yazıyı Console.WriteLine(); kullanarak yazarız.

    • Console.WriteLine(); : Alt alta yazıldığında çıktı da alt alta olur. Console.Write(); methodunun etkisini bitirmek içinde kullanılır.

  • Kullanım şekilleri aşağıda verilmiştir. 
    
class Program
{
    static void Main(string[] args)
    {
        // Yazi yazdırma (string)

        string yazi1 = "Merhaba ";
        string yazi2 = "Dunya ";

        // Write methodu ile yan yana yazdırıp,

        Console.Write(yazi1);
        Console.Write(yazi2);

        // WriteLine methodu ile yan yana yazdırmayı kapattık aynı işlem aşağıdada uygulanmıştır.

        Console.WriteLine("!");

        Console.Write(yazi1);
        Console.WriteLine(" Ekleme");

        // Yazi yazdırma (char)
        // Ekrana harf yazdırmanın yazı yazdırmaktan bir farkı yoktur.

        char harf = 'A';

        Console.Write(harf);

        Console.WriteLine('A')

        // Sayi yazdırma

        // Ekrana sayı yazdırmak için  2 yöntem vardır
        // İlk yöntem direkt sayı yazdırma  

        Console.WriteLine(1);

        // İkinci yöntem ise değişken tanımlayıp yazdırma

        int a = 1;
        Console.WriteLine(a.ToString());

        // Eğer bu şekilde yaparsanız '.ToString()' ifadesini kullanmak zorundasınız yoksa hata alırsınız.

        Console.ReadKey();

        // Bu ifadeyi kullanmazsanız program çalıştığı gibi kapanır çünkü ; tüm kodlar çalıştıktan sonra çalışacak başka kod kalmaz. 'Console.ReadKey();' methodu tuş olaylarını dinler.
        // Yani bir tuşa basana kadar bekler. Tuşa bastıktan sonra başka kod kalmayacağı için program kapanır. 
    }
}
