Değişkenler

  • C# da en sık kullanılan değişkenker : int, string, char

    • int : Sayı tanımlamak için kullanılır.

    • string : Yazı tanımlamak için kullanılır, değer atanırken " " kullanılır.

    • char : Harf tanımlamak için kullanılır, değer atanırken ' ' kullanılır.

  • Değişkenlerin kullanılabilmesi için bir isim vererek tanımlanması gerekir, aynı ismi 1 den fazla değişkene veremezsiniz.

  • Kullanım şekilleri aşağıda verilmiştir. 
    
class Program
{
    static void Main(string[] args)
    {
        string degisken_adi1 = "Bu bir yazı";

        int degisken_adi2 = 1;

        char degisken_adi3 = 'A';

        //İlk başa değişken türü, ardından degisken adı sonrasında '=' ve karşısına değişkenin değeri ve en sonada ';' kullanıyoruz.
        //Değeri illa böyle tanımlamak zorunda değiliz mesela böylede yapabiliriz :

        string yazi;
        yazi = "Bu da bir deger atamadır";
        char harf;
        harf = 'A';
        int sayi;
        sayi = 1;

        //Sona ';' koymazsanız hata verecektir.
    }
}
